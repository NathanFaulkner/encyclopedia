In the section entitled Systems of Equations we investigated the possible number of solutions to a system of two linear equations in two variables (\(x\) and \(y\)). There, we approached the question from a graphical perspective, which easily earned us the insight that the possible number of solutions to such a system is 0, 1, or \(\infty\). That's all! In this section, we learn a master-strategy (or algorithm, really) for finding the solution set. This algorithm will find the single, unique solution to the system if it has one but also identify whether the system either has no solution or an infinite set of solutions all lying along a line. So, we are switching perspectives—from a graphical perspective to an algebraic one, where we just manipulate symbols. Having these multiple perspectives (as well as a third: an intuitive grasp or mental representation of the meaning of the symbols) is really the key to mastering math. Each perspective has its strengths. The strength of the algebraic perspective is that it is so very definite. It is where you exercise careful, unambiguous logic, so that you can be certain of your results! In particular, you can check all of the answers you obtain in this section!

The particular algorithm or approach we are learning in this section, as you can see from the title, is called the elimination method. It is best learned by seeing it demonstrated. So, here goes!

Solve the following system using the method of elimination. That is, find a nice description of its solution set. \[ \begin{cases} x - 2y = -9 \\ 2x + 3y = 10 \end{cases} \]

Answer:

What we are doing when we find the solution to a system is assuming \(x\) and \(y\) are actual, honest real numbers with the property that when they are simultaneously substituted into each equation both equations are satisfied. So, the point is, by assumption, \(x\) and \(y\) represent the same number in each equation. Elimination uses that fact in a clever way: Note that if the two equations above are true, then so are these two: \[ \begin{cases} -2x +4y = 18 \\ 2x + 3y = 10 \end{cases} \] All I did was multiply both sides of the first equation by \(-2\). Why did I do that? It's so that I can now add the two left sides of the equations together and eliminate \(x\). Since the left sides equal the right sides, I know the sum of the left sides will equal the sum of the right sides. To be clear, since \(x\) and \(y\) are (by assumption) real numbers—meaning, they could actually be plotted on the real line somewhere!—I know, too, that \( -2x + 4y \) and \( 2x + 3y \) both represent real numbers. They are sumbols for real numbers. Actually, in this case I know a bit more than that! We are assuming that—even though I don't know what \(x\) and \(y\) are—I know that \(-2x + 4y\) is literally 18 and \(2x + 3y\) is literally 10. Put another way, I know something about what \(x\) and \(y\) do, but not yet what they are. The question is, do I know enough about what \(x\) and \(y\) do to figure out what they are?

Back to the strategy: I know that when I add 10 to 18, I will undoubtedly get 28. But have special representations of the numbers 10 and 18. I can also "add" those representations of 10 and 28 and simplify. That is, I "add the left sides together",, knowing that, when I do so, there sum will definitely be 28: \begin{align*} \phantom{+\quad} -2x +4y &= 18 \\ + \quad 2x + 3y &= 10\\ \hline 7y &= 28 \end{align*} Of course, from the equation that resulted, I quickly learn that \(y = 4\). That's the real number that \(y\) actually represented all along! It's just that it took a few steps to discover that. Next, remember out goal: by assumption, I am looking for an \((x,y)\)-pair that makes both equations true. Now that I know that the only possible \(y\)-coordinate for such a pair is 7, I can just plug in \(y=4\) to either equation to discover what \(x\) is supposed to be: \begin{align*} & -2x + 4\cdot 4 = 18 \\ \Rightarrow \quad & -2x + 16 = 18 \\ \Rightarrow \quad & -2x = 2 \\ \Rightarrow \quad & x = -1 \\ \end{align*} So, it looks like my solution set contains just the single pair \((-1,4)\). You can check your answer by plugging back in to make sure each equation is satisfied (or, at least, plug back into the other equation that you didn't just use to get the final answer): \[ -2\cdot(-1) + 4\cdot 4 = 2 + 16 = 18 \] and \[ 2 \cdot (-1) + 3\cdot 4 = -2 + 12 = 10 \] So, I know my answer is correct.

I will do the next examples faster—without quite so much explanation. The next example illustrates a case where you need to adjust both equations, not just one, to perform elimination.

Solve the following system using the method of elimination. That is, find a nice description of its solution set. \[ \begin{cases} 2x - 3y = 1\\ -3x + 7y = 1 \end{cases} \]

Answer:

To perform elimination on \(x\), you could multiply both sides of the first equation by \(3\) and both sides of the second equation by \(2\): \[ \begin{cases} 6x - 9y = 3\\ -6x + 14y = 2 \end{cases} \] Now, add the sides together: \begin{align*} 6x - 9y &= 3\\ + \quad -6x + 14y &= 2\\ \hline 5y &= 5 \end{align*} So, \(y=1\). Sub-in \(y=1\) to either equation to get the value of \(x\). I used the first equation, from the original system: \begin{align*} & 2x - 3\cdot 1 = 1\\ \Rightarrow \quad & 2x - 3 = 1 \\ \Rightarrow \quad & 2x = 4 \\ \Rightarrow \quad & x = 2 \\ \end{align*} So, the solution set just contains \((2,1)\). Remember, you can check your answer by plugging back in and seeing if this pair satisfies both equations! It does: \[ \begin{cases} 2x - 3y = 2\cdot 2 - 3\cdot 1 = 4 - 3 = 1\\ -3x + 7y = -3\cdot 2 + 7\cdot 1 = -6 + 7 = 1 \end{cases} \]

Now, what happens in the elimation procedure when the solution set is empty or consists of an infinite set of points along a single line? That's what the next two examples are designed to explore.

Solve the following system using the method of elimination. That is, find a nice description of its solution set. \[ \begin{cases} x - 2y = -9 \\ -4x + 8y = 36 \end{cases} \]

Answer:

To perform elimination on \(x\), you could multiply the first equation by \(4\). Here is what results: \[ \begin{cases} 4x - 8y = -36 \\ -4x + 8y = 36 \end{cases} \] When you add the sides together, though, you get \[ 0 = 0 \] Strictly speaking, that tells you nothing. I mean, we already knew that 0 equals 0 with doing all that work. But we can take it as a reliable indicator that when you graph the two equations, you get the same line. Just look at the two equations! They only differ by a factor of \(-1\)! So, the solution set is infinite, consisting in all points along the line with equation \(x - 2y = -9\).

A mathematician would represent this set using "set notation". Do remember the notation that used "curly braces" that we discussed in the section on function notation? There, we defined functions as a special set of ordered pairs. Well, the solution to a system of equations is a set of ordered pairs as well (though you wouldn't necessarily think of it as a function). So, like in the other section I just linked to, we can use set notation to represent it. Previously, we used set notation to just provide a scope for listing elements in a set. For instance, \[ \{(1, 2), (-5, 6), (9,12)\} \] is a set of ordered pairs. But the solution set for this example is an infinite set, so this notation isn't entirely adequate. We need to adapt it in a certain way. Specifically, as there is now way to list an infinite set of this nature, we have to describe the set by a "rule". This is much the same as what we did when we used function notation instead of set notation to define a set. But, really, you can use both. Here is a symbol a mathematician would use to represent the solution set for this system: \[ \left\{(x, y)\mathbin{|} x - 2y = -9\right\} \] What does this mean? Well, here is how you would "read" it:
\(\{\) \((x, y)\) \(|\) \(x - 2y = -9\) \(\}\)
"Set of" \((x,y)\)-pairs "with the property that" "this equation is true" [end set]
So to be clear, it is a symbol for "the set of all \((x, y)\)-pairs that satisfy the equation \(x - 2y =-9\)"—which is none other than the solution set for this system. Now, this notation is kind of clunky, so on the exercise page, you can just enter the equation part. You can enter either equation as a description of your set of infinite solutions when that case comes up.

Solve the following system using the method of elimination. That is, find a nice description of its solution set. \[ \begin{cases} x - 2y = -9 \\ -4x + 8y = 30 \end{cases} \]

Answer:

To perform elimination on \(x\), you could multiply the first equation by \(4\). Here is what results: \[ \begin{cases} 4x - 8y = -36 \\ -4x + 8y = 30 \end{cases} \] When you add the sides together, though, you get \[ 0 = -6 \] Of course, that is a false statement. That means there are no such \(x\) and \(y\) that make both equations true: There is no solution.
