For each of the following parts, consider this inequality. \[ 3x - 7 \lt 8 \]
  1. Solve the inequality.

    Answer:

    As completed in solving linear inequalities , we can solve as follows: \begin{align*} & 3x-7 \lt 8 & \textrm{Given}\\ \Rightarrow \quad & 3x \lt 15 & \textrm{I added 7 to both sides.}\\ \Rightarrow \quad & x \lt 5 & \textrm{I divided both sides by 3.}\\ \end{align*}
  2. Graph this solution on the real line.

    Answer:

    Now for the new part! The first thing to observe is that you are shading (in this case in blue) to indicate what \(x\)-values are part of the solution set. But since all values up to but not including \(x=5\) are part of the solution set, we need a marker at that point—but a special marker to communicate to your reader that that point is not actually part of the solution set. An "empty circle" is that symbol. In the next example, you will see that a "filled circle" when this boundary point is actually part of the solution set (which, I hope you will realize, happens when an "or equal" inequality is employed.) {% set svg = 'svg1' %} {% set points = [{'x': 5, 'type': 'empty'}] %} {% set intervals = [[-20,5]] %} {% include "_static-real-line.html" %}

Solve the inequality. Graph the solution. \[ - 12 x - 6 \leq - 8 x \]

Answer:

\begin{align*} & - 12 x - 6 \leq - 8 x & \textrm{Given}\\ \Rightarrow \quad & - 4 x - 6 \leq 0 & \textrm{I added \(8x\) to both sides.}\\ \Rightarrow \quad & -4x \leq 6 & \textrm{I added 6 to both sides.}\\ \Rightarrow \quad & 4x \geq -6 & \textrm{I negated both sides. (Reverse the inequality!}\\ \Rightarrow \quad & x \geq \frac{-6}{4} & \textrm{I divided by 4.}\\ \Rightarrow \quad & x \geq -\frac{3}{2} & \textrm{I simplified.}\\ \end{align*}

Now, we graph it. This time, I will use a "filled" point since \(-\frac{3}{2}\) is part of the solution set (since \(x\) is allowed to equal it as per the "or equal to" part of the greater-than-or-equal-to sign). Also, because the solution set includes points greater than the boundary value of \(-\frac{3}{2}\), I will shade to the right: {% set svg = 'svg2' %} {% set points = [{'x': -3/2, 'type': 'filled'}] %} {% set intervals = [[-3/2,20]] %} {% include "_static-real-line.html" %}
